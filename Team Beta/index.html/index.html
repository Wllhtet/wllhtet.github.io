Images

images serve as visual representations that showcase a company's products and services Images can be photos, drawings, charts, diagrams and other graphics to contribute to the design and asthetic of the webpage. when web designers include images on a website, they must take into account factors like the appropriate file format, dimensions, and file size. This consideration ensures that the images are optimized for web use, allowing for faster loading times, better user experience, and efficient use of bandwidth.

Here are the several formats that can be use for appropriate design

  • GIF - Graphics Interchange Format and is pronounced as “jiff”. GIF is the oldest web file format and supports transparency and frame animation
  • PNG - Portable Network Graphic and is pronounce as “ping”. PNG utilizes lossless compression and can accommodate 8-bit color, 16-bit grayscale, and 24-bit true-color images, making it versatile for various types of images while maintaining image quality.
  • JPG - Joint Photographic Experts Group and is pronounced as “jay-peg”. JPG is ideal for high-detail images like digital photos due to its support for 16.7 million colors in a 24-bit format, making it perfect for capturing intricate details and complex color compositions, including shadows.
  • SVG - Scalable Vector Graphics (SVG) is a file format that employs markup language to generate two-dimensional graphics, images, and animations, offering scalability without loss of quality.Scalable Vector Graphics (SVG) is a file format that employs markup language to generate two-dimensional graphics, images, and animations, offering scalability without loss of quality.
